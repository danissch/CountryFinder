# CountryFinder
This project is a demostration for iOS getting data of countries from https://rapidapi.com/apilayernet/api/rest-countries-v1 API, so you can search and select a country to see more details about location.

## Installation

### Usage
Clone, open and compile the project in simulator or device. Make sure you did the authorize in your device to run the app, all is by the Apple restrictions to run apps out of the AppStore). 
This app may run in iOS +13
